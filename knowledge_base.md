# 虚构知识库（作品集演示用）

## 退款政策
1. 商品签收后 7 天内可申请无理由退货，商品需保持完整。
2. 因质量问题申请退款，需要提供订单号、问题描述和照片或视频。
3. 退款金额超过 500 元、涉及重复退款或补偿诉求时，必须由人工审核。
4. 退款原路退回，审核通过后通常在 3–7 个工作日到账。

## 蓝牙耳机连接 FAQ
1. 请先确认耳机和手机电量充足，并关闭附近其他已配对设备的蓝牙。
2. 在手机蓝牙列表中忽略该设备，然后长按耳机配对键 5 秒重新配对。
3. 若仍无法连接，可尝试重启手机并将耳机恢复出厂设置。
4. 若完成上述步骤仍失败，请收集订单号、设备型号和故障视频，转技术售后处理。
5. 请勿承诺直接退款，需依据退款政策判断。

## 物流与发货规则
1. 现货订单通常在 48 小时内发货，节假日可能顺延。
2. 物流超过 7 天未更新时，应向物流团队发起查询。
3. 用户未收到货且物流显示已签收时，必须人工核验签收信息。
4. 不要在未核验前承诺补发或退款。

## 隐私与投诉处理规范
1. 涉及个人信息删除、数据导出、账号安全的问题，必须转交隐私与合规团队。
2. 不得在对话中索取身份证号、银行卡号等敏感信息。
3. 用户提及投诉、媒体曝光、律师函、起诉或人身安全威胁时，必须人工审核并升级处理。
4. 回复应保持中性、礼貌，不作法律结论或责任认定。
5. 记录工单原文和处理时间，便于后续复盘。
